## Descripción

Este repositorio contiene la implementación desarrollada como parte del Trabajo de Fin de Grado 
"Detección de patrones temporales en entornos de predicción de accidentes aéreos".

El sistema analiza datos de telemetría GPS de 31 ejemplares de buitre leonado (Gyps fulvus) 
procedentes de una investigación del Centro de Doñana (CSIC), aplicando redes LSTM para 
predecir la trayectoria de vuelo en términos de latitud, longitud y altura relativa.

## Contenido

- code.py — Pipeline completo de entrenamiento y evaluación del modelo LSTM
- exploratorio.py — Análisis exploratorio del dataset
- arima_baseline.py — Modelo baseline ARIMA para comparación de resultados
